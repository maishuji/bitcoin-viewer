# Bitcoin viewer
Early project aiming to show bitcoin data with diagrams

## Prerequisites

Before you begin, ensure you have met the following requirements:

- **CMake** (version 3.10 or higher)
- **A C++ Compiler** (GCC, Clang, MSVC, etc.)
- **Python** (if applicable, for dependencies like NumPy)
- **nlohmann-json** (sudo apt install nlohmann-json3-dev on Ubuntu/Debian)
- **CURL** (sudo apt install libcurl4-openssl-dev on Ubuntu/Debian)

## Build

```bash
mkdir build
cd build
cmake ..
```

## Running the Program

```bash
./bitcoin_plot
```
